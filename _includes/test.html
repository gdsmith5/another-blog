This section calculates the Taylor series expansion for e^x

taylor = 0;
lim = 10;
x = 0.2;

% calculate the Taylor series expansion for e^x using 'lim' number of terms
for i = 0:lim
   taylor = taylor + x^i/factorial(i);
end
